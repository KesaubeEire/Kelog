重申七项基本原则

  1. 第一,保证自己的核心竞争力
    • 游戏是你始终不变的标签,不但不能荒废,更加应该第一时间Update
    • 作为游戏的三大支柱,程序\Unity始终是自己的家伙饭,请千万不要忘记
    • 保持抽象的思维,对游戏\产品的敏感性需要长期保持,有一个产品经理的野心
    • 培养负担得起的审美能力,将来有机会尝试和自己的伴侣或者下一代共同培养艺术细菌
  2. 第二,保证自己的专注程度
    • 通过自己的意识反馈自己的专注程度,不要be distracted easily
    • 秋叶大叔,锻炼自己的专注力,认识到 专注程度 * 专注时间 才是输出价值的关键
    • 认真对待生物方面的作业和实验,即便你往后的人生中不会再次出现这些,也不应轻视
  3. 第三,维持走出舒适区的状态
    • 始终不以在别人面前装逼为价值,这是真正的不成熟,就是傻逼
    • 始终不以自己已经十分熟练的技能或成就的再次熟练操作而骄傲
    • 始终向有毅力,能坚持,肯谦虚,重成长的人学习
    • 始终告诫自己轻结果,少捷径,多钻研,勤思考,避讳快而不稳
  4. 第四,坚持合理的作息和锻炼
    • 回到每周4次的健身锻炼中,不以自己的日常作为而转移,不以自己的学业考试为借口
    • 每天早睡半小时,每天早起20分(从凌晨两点\早晨8点半为例) 从现在开始
    • 坚持正常的饮食时间和饮食搭配,少米多肉,少沙县多食堂
  5. 第五,保证英语的有效学习
    • 托福的OG,萌石这里有资源,千万不要忽视
    • 流利说的,每天签到,干起来吧,不给自己找退路
    • 听力:TeacherGwen 公众号
  6. 第六,养成几件受益终身的习惯
    • 脱离手机阅读,养成系统培养知识的习惯
      • 总结秋叶大叔的书单,开始买书,别想着省钱
      • 总结完成之后在碎片时间随手阅读,可以找个时间在整个的地忒上读书,我发现对我来说地铁是个非常好的读书地
    • 尝试控制自己的碎片时间,制作按照秋叶大叔所说的列一个各种时间间隔我应该做什么的表,比如1分钟\3分钟\5分钟\10分钟^
    • 坚持回到宿舍联系尤克里里,尝试,嗯,一个月一首新曲子,本学期还剩3首
  7. 最后,争取在今年养成”困难的事情优先做,恶心的事情第一做”的习惯
