  • 18.3.16

    • 继续 jQuery 简略学习

    • 选择器

      • 元素选择器：$(“button”)
      • class选择器：$(“.btn”)
      • id选择器：$(“#target1”)

      • 多项选择

        • $(“.target(class名称,id尚未试过,下同):nth-child(n)“)
          CSS选择器允许你按照索引顺序(从1开始)选择目标元素的所有子元素。
          我的理解是看到和名称相同的就开始计数,当父物体结束后计数归零,直到再次遇到名称相同的类
        $(".target:nth-child(2)").addClass("animated bounce");
        • $(“.target:nth-child(2)“)
          $(“.target:nth-child(2)“)
          选择奇数个或者偶数个
        $(".target:nth-child(2)").addClass("animated bounce");
$(".target:even").addClass("animated shake");

    • 方法

      • .addClass(“class名”) 为选择器的结果添加class
      • .remove() 删除选择
        • like: $("#target4").remove();
      • .removeClass(“class名”) 为选择器的结果删除class

      • .css(“属性”,“值”) 更改选择器的结果的css的相应属性的值

        • like
        $("#target1").parent().css("background-color", "red")

      • .prop(“操作”,值)

        • 我们是这样来让按钮不可选的:
          • $("button").prop("disabled", true);

      • .html(“内容”) 添加HTML标签和文字到元素，而元素之前的内容都会被方法的内容所替换掉。

      • .text(“内容”) 把传进来的任何东西(包括标记)当成文本来显示,同样替换掉原来标签里的内容

      • .appendTo(” 目标id或class “) 将被选择移动到目标class或id下

      • .clone() 复制某被选择

      • .clone().appendTo() 复制被选择到目标class或id下

        • 这样连起来玩叫做 方法链 function chain

      • .parent() 访问父元素,非常6

      • .chlidren() 访问子元素,更是6
