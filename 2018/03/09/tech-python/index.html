  • 可变参数 & 关键字参数
    • 可变参数允许你传入0个或任意个参数，这些可变参数在函数调用时自动组装为一个tuple。
    • 关键字参数允许你传入0个或任意个含参数名的参数，这些关键字参数在函数内部自动组装为一个dict
    • 关键在于理解 *list 和 **dict 都是干嘛的
  • 给函数换名字,直接赋函数名给想要替换的另一个函数名,无需 “”,真TM牛逼

  • 可变对象 和 不可变对象

    • 对于可变对象的操作是直接作用在对象本身的
    • >>> a = ['c', 'b', 'a']
>>> a.sort()
>>> a
['a', 'b', 'c']

    • 而对于不可变对象的操作对对象本身没有影响
      而是返回一个复制值再进行操作

    • >>> a = 'abc'
>>> b = a.replace('a', 'A')
>>> b
'Abc'
>>> a
'abc'

    • 可变对象有: list,dict
      不可变对象有:int,string,float,tuple

      • list & dict 的区别 dict用空间换时间
        • 和list比较，dict有以下几个特点：

        • 而list相反：
          • 查找和插入的时间随着元素的增加而增加；
          • 占用空间小，浪费内存很少。

    • # 比如定义一个函数，包含上述若干种参数：
def f1(a, b, c=0, *args, **kw):
  print('a =', a, 'b =', b, 'c =', c, 'args =', args, 'kw =', kw)

def f2(a, b, c=0, *, d, **kw):
  print('a =', a, 'b =', b, 'c =', c, 'd =', d, 'kw =', kw)
#
# 在函数调用的时候，Python解释器自动按照参数位置和参数名把对应的参数传进去。
#
>>> f1(1, 2)
a = 1 b = 2 c = 0 args = () kw = {}
>>> f1(1, 2, c=3)
a = 1 b = 2 c = 3 args = () kw = {}
>>> f1(1, 2, 3, 'a', 'b')
a = 1 b = 2 c = 3 args = ('a', 'b') kw = {}
>>> f1(1, 2, 3, 'a', 'b', x=99)
a = 1 b = 2 c = 3 args = ('a', 'b') kw = {'x': 99}
>>> f2(1, 2, d=99, ext=None)
a = 1 b = 2 c = 0 d = 99 kw = {'ext': None}
>>> args = (1, 2, 3, 4)
#
# 最神奇的是通过一个tuple和dict，你也可以调用上述函数：
#
>>> kw = {'d': 99, 'x': '#'}
>>> f1(*args, **kw)
a = 1 b = 2 c = 3 args = (4,) kw = {'d': 99, 'x': '#'}
>>> args = (1, 2, 3)
>>> kw = {'d': 88, 'x': '#'}
>>> f2(*args, **kw)
a = 1 b = 2 c = 3 d = 88 kw = {'x': '#'}
      • 关键字参数
        • 关键字参数允许你传入0个或任意个含参数名的参数，这些关键字参数在函数内部自动组装为一个dict
    • def person(name, age, **kw):
            print('name:', name, 'age:', age, 'other:', kw)
###
# 函数person除了必选参数name和age外，还接受关键字参数kw
# 在调用该函数时，可以只传入必选参数：
>>> person('Michael', 30)
name: Michael age: 30 other: {}
###
# 也可以传入任意个数的关键字参数：
>>> person('Bob', 35, city='Beijing')
name: Bob age: 35 other: {'city': 'Beijing'}
>>> person('Adam', 45, gender='M', job='Engineer')
name: Adam age: 45 other: {'gender': 'M', 'job': 'Engineer'}
###
# 若是使用更简化的方法呢
>>> extra = {'city': 'Beijing', 'job': 'Engineer'}
>>> person('Jack', 24, **extra)
name: Jack age: 24 other: {'city': 'Beijing', 'job': 'Engineer'}
      • 命名关键字参数
        • 用以上的方法,关键词参数不受限制,但是我们若想知道有哪些关键字
      • def person(name, age, **kw):
      if 'city' in kw:  # 有city参数
        pass
      if 'job' in kw:   # 有job参数
        pass
      print('name:', name, 'age:', age, 'other:', kw)
###
# 但是此时仍然没有限制关键字参数
>>> person('Jack', 24, city='北京', addr='海淀', zipcode=034)
        • 如果要限制关键字参数的名字，就可以用命名关键字参数，例如，只接收city和job作为关键字参数。这种方式定义的函数如下：
      • def person(name, age, *, city, job):
  print(name, age, city, job)
###
# 调用方法如下
###
>>> person('Jack', 24, city='Beijing', job='Engineer')
Jack 24 Beijing Engineer
###
# 命名关键字参数必须传入参数名，这和位置参数不同。如果没有传入参数名，调用将报错：
###
>>> person('Jack', 24, 'Beijing', 'Engineer')
Traceback (most recent call last):
TypeError: person() takes 2 positional arguments but 4 were given
# 由于调用时缺少参数名city和job，Python解释器把这4个参数均视为位置参数，但person()函数仅接受2个位置参数。
# 但是你要是不打那两个key-word,也不设置默认值,也是不识别的
TypeError: person() missing 2 required keyword-only arguments: 'city' and 'job'
###
###
# 命名关键字参数可以有缺省值，从而简化调用：
###
def person(name, age, *, city='Beijing', job):
  print(name, age, city, job)
# 由于命名关键字参数city具有默认值，调用时，可不传入city参数：
>>> person('Jack', 24, job='Engineer')
Jack 24 Beijing Engineer
###
# 使用命名关键字参数时，要特别注意，如果没有可变参数，就必须加一个*作为特殊分隔符。如果缺少*，Python解释器将无法识别位置参数和命名关键字参数：
###
def person(name, age, city, job):
    # 缺少 *，city和job被视为位置参数
    pass

    • 定义函数

      • 可变参数

        • 数量可变是先搞一个list 或 tuple,然后把元素直接代成 list 或 tuple, 直接元素任然是 list 或 tuple
        •  def calc(numbers):
    sum = 0
    for n in numbers:
        sum = sum + n * n
    return sum

        • 定义可变参数和定义一个list或tuple参数相比，仅仅在参数前面加了一个*号。在函数内部，参数numbers接收到的是一个tuple，因此，函数代码完全不变。但是，调用该函数时，可以传入任意个参数，包括0个参数：

        • def calc(*numbers):
    sum = 0
    for n in numbers:
    sum = sum + n * n
    return sum

        • 如果已经有了一个list或者tuple,但是要调用一个可变参数要怎么办,可以这样

        • >>> nums = [1, 2, 3]
>>> calc(nums[0], nums[1], nums[2])
14
# 但是这样太繁琐了,于是我们又可以这样
>>> nums = [1, 2, 3]
>>> calc(*nums)
14

      • 默认参数 直接等于进去

        • 但是默认参数必须是不变参数,不能使list 和 dict
        • 设计不变对象的原因
          • 因为不变对象一旦创建，对象内部的数据就不能修改，这样就减少了由于修改数据导致的错误。
          • 此外，由于对象不变，多任务环境下同时读取对象不需要加锁，同时读一点问题都没有。
          • 我们在编写程序时，如果可以设计一个不变对象，那就尽量设计成不变对象。

      • 返回多个值 , 用逗号隔开不同的返回值

        • 其实是假的,是用tuple封装了一下,其实返回的还是一个对象

      • 空函数 pass

        • 不做任何操作,只是占一个位置
        • 有的地方缺了 pass 会有错误(比如 if x:的下面)
