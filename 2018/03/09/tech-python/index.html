不会的重点

  • 可变对象 和 不可变对象

    • 对于可变对象的操作是直接作用在对象本身的
    • >>> a = ['c', 'b', 'a']
>>> a.sort()
>>> a
['a', 'b', 'c']

    • 而对于不可变对象的操作对对象本身没有影响
      而是返回一个复制值再进行操作

    • >>> a = 'abc'
>>> b = a.replace('a', 'A')
>>> b
'Abc'
>>> a
'abc'

    • 可变对象有: list,dict
      不可变对象有:int,string,float,tuple

  • dict 和 set

    • set 简单粗暴的Hashset

      • api

        • set_object.add(element) 添加
        • set_object.remove(element) 删除
        • 集合型操作:交集 & 并集全部集合型操作
        • # 部分操作,全部详见上面的链接
a = t | s    # t 和 s的并集  
b = t & s    # t 和 s的交集  
c = t – s    # 求差集（项在t中，但不在s中）  
d = t ^ s    # 对称差集（项不同时在s和t中）

        • 啊哈

    • dict 键值对

      • 表示:
    • # 用list实现
names = ['Michael', 'Bob', 'Tracy']
scores = [95, 75, 85]
# 用dict实现1
>>> d = {'Michael': 95, 'Bob': 75, 'Tracy': 85}
>>> d['Michael']
95
# 用dict实现2
>>> d['Adam'] = 67
>>> d['Adam']
67
###############
# 判断Key是否存在1
>>> 'Thomas' in d
False
# 判断Key是否存在2
>>> print(d.get('Thomas')) # 不存在的话print显示None
None
# 判断Key是否存在3
>>> d.get('Thomas', 233) # 这是自定义的不存在的显示情况
233                      # 逗号之后是自定义的显示内容
>>> d.get('Michael', 233) # 这是自定义的不存在的显示情况
95                        # 存在的话返回 键 对应的 值
      • list & dict 的区别 dict用空间换时间
        • 和list比较，dict有以下几个特点：
          • 查找和插入的速度极快，不会随着key的增加而变慢；
          • 需要占用大量的内存，内存浪费多。
        • 而list相反：
          • 查找和插入的时间随着元素的增加而增加；
          • 占用空间小，浪费内存很少。
      • dict_object.pop(key) 删除一个键值对

  • 循环

    • range函数 包前不包后
      • range(5) = [0, 1, 2, 3, 4]
      • range(4,10) = [4,5,6,7,8,9]
    • break & continue 的存在

  • list & tuple

    • list的API(无序列表,随你变,用 (&&) 作为括号)
      • list[0]~list[length-1] 和 list[-1]和list[-length]访问相同
      • list.append(“content”)添加到最后
      • list.insert(index,“content”)添加到指定索引
      • list.pop()删除最后
      • list.pop(index)删除指定

    • tuple的API(有序元组,不可变,用 [&&] 作为括号)

      • 定义 t = ()
      • 定义容量 t = (len,) | t = (len) 会被认为是只含数 len 的 tuple

      • 可变&不可变:tuple的元素指针指向的值是不变的,但当元素出现list时,list的子元素可以变

      • >>> t = ('a', 'b', ['A', 'B'])
>>> t[2][0] = 'X'
>>> t[2][1] = 'Y'
>>> t
('a', 'b', ['X', 'Y'])

      • 啊哈

  • 字符串方面

    • Python提供了 ord()函数获取字符的整数表示，chr()函数把编码转换为对应的字符

    • >>> ord('A')
65
>>> ord('中')
20013
>>> chr(66)
'B'
>>> chr(25991)
'文'
>>> '\u4e2d\u6587' # 懂十六进制的话可以这样
'中文'
# Python对bytes类型的数据用带b前缀的单引号或双引号表示：
x1 =  b'ABC'      # 字节类型
x2 =  'ABC'       # 字符串类型

    • 格式化:1.和c一模一样 2.format方法

    • # 1.和c一模一样
>>> 'Hello, %s' % 'world'
'Hello, world'
>>> 'Hi, %s, you have $%d. Give me %.2f' % ('Michael', 1000000,0.01)
Hi, Michael, you have $1000000. Give me 0.01
# 2.format方法
>>> 'Hello, {0}, 成绩提升了 {1:.1f}%'.format('小明', 17.125)
'Hello, 小明, 成绩提升了 17.1%'
