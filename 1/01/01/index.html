重点

  • 记一个 css 样式优先级的顺序

    • 最弱: 最上面个的 css
    • next: 往后的 css ,若 class 均在同一个对象中出现(与放置顺序无关),越往后的会覆盖掉前面的
    • next: id id 永远比 class 优先级高

    • next: 行内 style , 这个比前两者都高

      <style>
body {
background-color: black;
font-family: Monospace;
color: green;
}
.pink-text {
color: pink;
}
.blue-text {
color: blue;
}
#orange-text{
color:orange;
}
</style>
<h1 class="pink-text blue-text" id="orange-text">
Hello World!
</h1>

      Hello World!

    • final: 若在某对象中 class 有多个,其中的某一个有某属性后带了 !important,那它最最高

      <style>
#orange-text {
color: orange;
}
.pink-text {
color: pink !important;
}
.blue-text {
color: blue;
}
</style>
<h1 id="orange-text" class="pink-text blue-text" style="color: white">Hello World!</h1>

      Hello World!
